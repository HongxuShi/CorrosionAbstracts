# 项目任务与要求

## 1. 字体要求
- **大题目**：宋体，5号，不加粗
- **一级标题**：宋体，4号
- **二级标题**：宋体，小4号

## 2. PPT 要求
- **总页数**：少于 20 页（建议 14 页左右）
- **PPT 内容结构**：
  1. 项目简介
  2. 项目解决的问题
  3. 数据源样式及数据量
  4. 数据清洗以及可视化
  5. 模型建立分析
  6. 参考文献
- **PPT 重点**：前三页必须包含数据数量和爬取网站说明。

## 3. 图表要求
- **数量**：不少于 3 张，且内容需各不相同。
- **规范**：画图前需先进行数据切片，严禁直接在 Excel 中画图。

## 4. 第十二周任务
- 完成项目流程图。
